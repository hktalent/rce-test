Tool Integration Wizard

Tool Description

Synopsis

Define some characteristics of the tool such as name and icon and give some information about yourself.

Usage

On this page, some characteristics of the tool for the integrated component are defined. Information for the tool user or graphical elements is optional.

Required fields:

-          Name: This field defines the name of the component. The name is very important, because the tool will be displayed in the workflow editor palette with this name. Note that the name must not contain special characters. The name will not be accepted if there is another locally created tool with the same name.

Optional fields:

-          Icon Path: An icon for the tool can be selected here. At best, it is larger than 32x32 pixels and has the same width and height. You can select to copy it into the configuration folder, so there is no absolute path link for it any more.

-          Group Name: This defines the group in the workflow editor palette in which the component will be shown.

-          Description: You can provide a description of your tool here, which can be seen by other users when configuring a workflow.

-          Name/E-Mail: You can provide some contact information about yourself in this field, for example if someone has questions or wants to provide suggestions.
