Tool Integration Edit Wizard

 

Choose Configuration

Synopsis

First page of the wizard to edit an existing configuration from the configuration folder or to load an existing configuration from disk.

Usage

With this wizard, you can edit an existing active or inactive integration configuration. Every found configuration is in the list and can be selected, but you can also load a valid configuration that was not found. The other pages of the wizard will be filled with the content of the configuration.

After updating your component, it could be necessary to reopen your workflow editor (to get the correct information in the properties tab), but if you execute it without doing that it will always use the updated values.

If you change something on the inputs or outputs, unfortunately at the moment you have to remove the instances of your tool in a configured workflow and put it back in.
