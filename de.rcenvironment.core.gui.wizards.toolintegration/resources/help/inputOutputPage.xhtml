Tool Integration Wizard

Inputs and Outputs

Synopsis

Define the inputs and outputs the component for the tool should have in an RCE workflow.

Usage

On this page, the inputs and outputs for the component are defined. Their behavior is the same as other inputs or outputs of other components, but they will always be static. That means no user can add new ones or edit the ones you define here. It is not mandatory to define inputs and outputs. If there are no inputs, the component will run once at workflow start and will be finished then.

As for the inputs, you can choose the data types File and Directory. If you do so, it is possible to define a file or directory name (File/Dir). Thereby the incoming file/directory will always be renamed to it.

If no value is entered, the file/directory will have its original name (noted by “As incoming”).
